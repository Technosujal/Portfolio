# Professional Portfolio Website

A modern, high-performance personal portfolio website designed to showcase projects, skills, and experience with a premium aesthetic.

## üöÄ Overview

This project is a fully responsive Full-Stack portfolio built using **Node.js** and **Express** for the backend, with a highly interactive frontend crafted in **HTML5**, **CSS3**, and **Vanilla JavaScript**.

The design features a trendy **Bento Grid** layout, smooth animations, and a fully functional **Dark/Light mode** toggle.

## ‚ú® Key Features

-   **Modern UI/UX**: Clean, grid-based layout (Bento Box style) with glassmorphism effects.
-   **Theme Support**: One-click toggle between Dark and Light themes.
-   **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.
-   **Dynamic Content**:
    -   **Interactive Hero Section**: Rotating text badges and entrance animations.
    -   **Project Showcase**: Detailed cards with hover effects and links.
    -   **Experience Timeline**: Vertical timeline for education and internships.
-   **Backend Integration**: Served via Express.js, ready for API expansions.
